# Cocktails API

REST API do zarządzania koktajlami i składnikami zbudowane w NestJS.

## Live

https://cocktails-api-production.up.railway.app/api

## Technologie

- NestJS + TypeScript
- PostgreSQL + TypeORM
- JWT autoryzacja
- Swagger dokumentacja

## Uruchomienie lokalne
```bash
npm install
npm run start:dev
```

## Testy
```bash
npm run test
npm run test:e2e
```
